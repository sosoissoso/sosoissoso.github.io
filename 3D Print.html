This was the first attempt of printing out my model.
I initially did not give the model any features to create any depth.
Nor did I make it thick enough as it ended up splitting in half.
I learned from these mistakes and made the necessary adjustments for the next printing.
As I watched the process of my model being printed, I noticed that the build up of
the newer model, each layer would have a honeycomb pattern to make each one stable.
After one layer was done printing, the machine would cover it with a
more fined layer as if it was covering up the layer.
